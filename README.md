## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

There are a lot of to-do lists but I don't see many where others can add in things needed. In my family, we are constantly adding things to grocery lists, so it would be easier if there was one place for everyone to add what they need.

Here's why:

- Everyone retains thoughts differently. If I get told someone needs something in passing, it could be forgotten.
- Families should be able to view everything on the list and make changes.

Use the `BLANK_README.md` to get started.
